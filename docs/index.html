Updating from an older version?

Smart Lower Thirds no longer uses a built-in local webserver and no longer supports scene binding. The overlay is now loaded as a local file in an OBS Browser Source, and lower thirds are triggered via dock controls, hotkeys, or timed automation.

Migration checklist: replace any http://127.0.0.1:.... URLs, update templates to the new placeholders, and review the install steps below.

How the overlay works

Smart Lower Thirds uses OBS Studio‚Äôs Browser Source with a local file path. This keeps setup simple and avoids background services.

Local file setup (recommended)

Create a Browser Source in OBS and point it to the plugin‚Äôs overlay HTML file (local file). Your lower thirds are rendered inside that page using your templates and settings.

  • No localhost URLs
  • No ports or firewalls
  • Easy to move between machines with your assets

If you are migrating from an older version, review the update notice in the hero section before replacing your Browser Source settings.

Dock Interface

Settings Dialog

Configure styling, animations, and timed behavior per lower third.

Updated placeholders

Templates are simple HTML/CSS files. Smart Lower Thirds injects your text, colors, and images at runtime using the placeholders below.

Available placeholders

Use these in your template HTML and CSS:

Placeholder Description
{{ID}} Unique instance ID (useful for per-item CSS targeting)
{{TITLE}} Main line (name, topic, headline)
{{SUBTITLE}} Secondary line (role, description, handle)
{{BG_COLOR}} Background color injected from settings
{{TEXT_COLOR}} Text color injected from settings
{{FONT_FAMILY}} Font family (defaults to Inter if not set)
{{PROFILE_PICTURE_URL}} Profile picture URL (local relative path). If no picture is set, this resolves to ./.

If you maintain older templates, update them to use {{PROFILE_PICTURE_URL}} and ensure your CSS references the new placeholder keys.

How to trigger lower thirds

Smart Lower Thirds focuses on reliable, client-facing controls: dock controls, hotkeys, and timed automation.

  • Dock: click to show/hide any lower third instantly
  • Hotkeys: map actions for fast live control (Stream Deck friendly)
  • Timed: auto-show every X seconds for Y seconds (set interval to 0 to disable)

Scene binding has been removed to keep the workflow consistent and predictable across OBS setups.

Installation

Install the plugin, enable the dock, then add a Browser Source using a local file overlay page.

Quick start

  1. Download the plugin for your platform from the GitHub releases page.
  2. Run the installer (or copy files into your OBS plugins folder).
  3. Restart OBS Studio.
  4. Enable the Smart Lower Thirds dock from the Docks menu.
  5. Create a Browser Source and set it to Local file, then browse to the Smart Lower Thirds overlay HTML file.
  6. Create your first lower third, then trigger it via the dock, hotkeys, or timed automation.
